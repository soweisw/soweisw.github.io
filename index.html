EECS 494 P1

Link: EECS 494 P1 Itch.io

This project uses Unity and C# to recreate the appearance of The Legend of Zelda (1986), as well as an additional level and mechanic. I did much of the work on UI, sprites, sound effects, music, and creating the overall look and feel of the game. Jackson Donaldson, my partner, did the real meat of the project, completing a large part of the game's physics.

EECS 494 P2

Link: EECS 494 P2 Itch.io

This project uses Unity and C#, and implements an interesting mechanic, in which the ability to see objects determines whether or not they have collision. Throughout this project, I learned much more about the C# Event system, Unity rendering filters, using the Unity Asset Store, and creating and utilizing non-Monobehaviour classes.

Night of the Living Ducks

Link: Night of the Living Ducks Itch.io

This project, the third project for EECS 494, uses Unity and C# to make a co-op multiplayer game in which rubber ducks fight off hordes of incoming zombie rubber ducks. I worked quite a bit on UI, sounds, weapon skins, and all in all making the game look and feel good. The most notable thing about my work on this project is the large amount of marketing material I created, from the Itch.io and Gamejolt pages, to the game's trailer, the latter of which I'm really proud of.
